## 页面元素说明

1. **商品轮播图**
   - 顶部展示商品的多张实物图
   - 支持左右滑动切换图片
   - 图片下方可有小圆点指示当前图片位置

2. **商品基本信息**
   - 商品名称
   - 商品价格
   - 销量信息

3. **商品详情介绍**
   - 文字描述：材质、尺码、颜色等基本特征
   - 图文详情：多张商品细节图片，使用场景图片
   - 尺码表或其他规格信息

4. **底部操作栏**
   - 收藏按钮
   - 加入购物车按钮
   - 立即购买按钮

5. **底部TabBar导航**
   - 固定在页面底部
   - 包含四个选项：首页、分类、购物车、我的
