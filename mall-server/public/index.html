商城小程序API服务

欢迎使用商城小程序API服务。本服务提供了完整的RESTful API接口，用于支持微信小程序商城和管理后台的开发。

查看API文档

API文档说明

API文档使用Swagger UI构建，提供了以下功能：

  • 所有API端点的详细说明
  • 请求和响应示例
  • 在线接口测试工具
  • 认证和授权方式说明

主要接口

  • 管理员接口: 提供管理员登录和账户管理功能
  • 用户接口: 提供用户注册、登录和个人信息管理
  • 商品接口: 提供商品信息查询和管理
  • 订单接口: 提供订单创建和管理
  • 购物车接口: 提供购物车管理

注意： API文档中的某些接口可能需要认证才能访问。请先获取有效的访问令牌。

健康检查

您可以通过访问 /api/health 来检查API服务的运行状态。
