静态资源目录

本目录用于存放商城小程序的静态资源文件，如CSS、JavaScript、字体等。

目录结构建议：
- css/: 存放样式表文件
- js/: 存放JavaScript文件
- fonts/: 存放字体文件
- images/: 存放通用图像文件（非上传的图片）

使用规范：
1. 静态资源文件名应使用小写字母、数字和连字符(-)组成
2. 为避免缓存问题，可以在文件名或URL参数中添加版本号
3. 生产环境应使用压缩后的资源文件

访问路径：
- 静态资源可以通过 /static/{文件路径} 路径访问
- 完整URL示例：http://localhost:8080/static/css/main.css

注意事项：
- 此目录不适合存放用户上传的文件，上传文件请使用 /uploads 目录
- 静态资源文件应定期优化以提高加载性能
- 建议使用CDN来分发静态资源
