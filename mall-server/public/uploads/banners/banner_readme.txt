轮播图文件存放规范

本目录用于存放商城小程序的轮播图图片文件。

命名规范：
1. 轮播图文件应该使用系统自动生成的名称，格式为：{时间戳}-{UUID}.{扩展名}
2. 不要手动更改文件名，以免引起系统无法正确加载图片

图片要求：
1. 推荐尺寸：750px × 375px (宽 × 高)，保持2:1的宽高比
2. 文件格式：支持 JPG, JPEG, PNG, GIF 等常见图片格式
3. 文件大小：单张图片不超过2MB

访问路径：
- 图片上传后，可以通过 /uploads/banners/{文件名} 路径访问
- 完整URL示例：http://localhost:8080/uploads/banners/1234567890-abcdef.jpg

注意事项：
- 仅上传与轮播图相关的图片文件
- 定期清理不再使用的轮播图文件
- 上传的图片应符合相关法律法规要求
